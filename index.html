Browsers block cross-origin API calls from file:// pages. If your API doesn't allow CORS from your origin, set a proxy URL here (e.g. https://corsproxy.io/?) and requests will be routed through it. Leave blank if your API already returns the right CORS headers, or if you run this page behind your own dev server.
Heads up: public proxies like corsproxy.io and allorigins.win only forward GET. For POST/PUT/PATCH/DELETE, either clear this field or host your own proxy.
